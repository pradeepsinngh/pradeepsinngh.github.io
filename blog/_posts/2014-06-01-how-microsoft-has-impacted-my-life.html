How Microsoft has impacted my life !!!!

“Microsoft” the one company that had made my life, and I may say a billion others, interesting with its enthusiastic and innovative ideas. Some other Product by Microsoft which made my life..!!

Microsoft Visual Studio :

Now, if you ask me which is the most powerful IDE for building applications is be it a Website, Web Application, or a Web Service, the answer would be incontrovertibly Microsoft Visual Studio, which is a stable standardized, mature, and secure environment.

If I have to point out an IDE, that enables me to write, debug, deploy and test applications, that would be MS Visual Studio as Integrating Visual Studio with Team Foundation Server and Project Server makes the  application development, deployment and testing  as easy as can be.
This is where I got connected to Microsoft because all this can be achieved seamlessly at one place, which is none other than
Microsoft Visual Studio.
And the Latest Visual Studio Pro Update 2 is like anything..!! Microsoft at its best.
Microsoft Office :
The most popular Microsoft Office is an office suite of desktop applications and services,which comes up with power point presentations which is favorite tool for most school going kids and of course Microsoft word which is an easy and near-instant textual input for processing of data

Microsoft Excel

It helps in many business organizations with storing,organizing and manipulating data and obviously with the help of Access large reports can be generated easily.

Internet Explorer 

This is something I am sure most of us will say when asked which was the first ever internet browser we used. And, I can bet on that. Having been the leading, or probably the only browser, used by millions worldwide for years, this browser sure did set the standard for the rest to come.

Outlook

 It is the personal email service which is developed by Microsoft and has provide a easy user interface to interact people all around the globe by sending emails quickly.It is much more than a simple boring inbox that we used to use before.

Skype

Now,keeping in touch with friends or family hasn’t been as difficult as it was a few years back.Microsoft introduced SKYPE in August 2003 and since then it has become most popular tool foe video conferencing among people situated anywhere in the world.

XBOX

And yes!How can we forget the  Xbox – Microsoft’s rendition on changing the gaming experience has indeed taken the virtual video game world by a storm. The Xbox has grown into the Xbox 360 and Xbox One. Giving the best gaming experience sitting in home and visualizing effects that are amazing.

Microsoft Dreamspark

 It is a Microsoft program originated mainly for the purpose of providing free tools and software designs to students.To register, students must visit the DreamSpark website and verify their identity.

Building on the partnership with Nokia announced in February 2011 and the increasing success of Nokia’s Lumia smartphones, Microsoft aims to accelerate the growth of its share and profit in mobile devices through faster innovation, increased synergies, and marketing.
